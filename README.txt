This program takes typing speed tests for you at inhumane speeds. There are two options for websites to chose from when the program is launched.

You may have to download a new chromedriver.exe depending on what version of chrome you use. You can download new versions here: https://chromedriver.chromium.org/

In case you have not done this already, open up a terminal console and navigate to the folder that holds the .py file. Then type "pip install selenium". Or, depending on your OS and/or python/pip version, do "pip3 install selenium".
